# Backend Wizards Stage 2 API

## Overview
This project is a Spring Boot API that classifies user data using external APIs and stores results in a database.

---

## Features
- Name classification using external APIs
- Gender detection (Genderize API)
- Age prediction (Agify API)
- Nationality detection (Nationalize API)
- Data persistence (Database)
- Filtering, pagination, sorting
- Clean structured API response (DTO)

---

## Endpoints

### Create Profile
POST /api/classify

### Get Profiles
GET /api/profiles

Supports:
- pagination
- sorting
- filtering

---

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL/MySQL
- REST APIs

---
